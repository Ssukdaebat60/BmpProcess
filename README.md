# BmpProcess

It is cpp Header to process the bmp image file.  

It provides functions such as...
 - open the bmp file
 - write the bmp file
 - delete dynamic array
 - getter of bmp information
 - select the area
 - brightness filter
 - color filter
 - mosaic
 - drawline
 - gray
 - binarization
 - reverse
 - contrast

  
## Requirements
- C++
